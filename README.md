# 重庆邮电大学本科毕业（论文）LaTeX版本使用手册
此文档用于重庆邮电大学本科毕业设计（论文）的 LaTeX 模板使用，LaTeX 是一种基于Tex的排版系统，LaTeX版本与Word版本在格式要求上完全相同。
## LaTeX 环境配置
推荐使用 texStudio 作为 LaTeX 的编辑器，安装了 texLive 才能使用 texStudio 编辑器，所以需要下载 texLive（必选）和 texStudio（可选）。
